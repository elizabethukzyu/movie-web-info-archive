# movie-web-info-archive
该仓库主要用于收集和归档影视相关网站的公开基础资料，涵盖电影、电视剧及综合影视内容站点。仓库内容仅限站点层级的信息整理，不包含任何影视内容文件或播放入口。所有资料整理自公开渠道，目的在于形成清晰、可长期保存的站点信息档案。本项目不参与站点运营，也不代表对所记录网站的认可或背书，仅作为中性的信息整理与留存工具。
